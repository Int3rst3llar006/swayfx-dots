swayfx-dots

a simple script that installs my dotfiles onto Arch

NOTE: you will have to do these things after installation. 
. change the ZSH_THEME variable to powerlevel10k
. change the config2 and style.css2 back to config and style.css in wofi directory
. set up powerlevel10k
. have to export the pywal16 configuration into .zshrc to make the configuration reload every time a new terminal opens.
.have to change the path to wallpaper to /home/YOUR_USERNAME/swayfx-dots/aesthetic-computer-4k.jpg

NOTE: you can change the wallpaper and colorscheme by replacing the path in swayfx config to the path to your new wallpaper and by running wal -i /home/YOUR_USERNAME/Downloads/your-wallpaper.format
